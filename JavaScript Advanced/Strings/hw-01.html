A va b, 2 ta string berilgan bo‘lsa, qisqa+uzun+qisqa ko‘rinishdagi satrni qaytaring, qisqaroq string tashqi tomondan, uzunroq string esa ichki tomonda. Satrlar bir xil uzunlikda bo'lmaydi, lekin ular bo'sh bo'lishi mumkin (uzunligi 0). Namunalar comboString('Hello', 'hi') → hiHellohi comboString('aaa', 'b') → baaab
